# ML Algorithms from Scratch

This repository contains implementations of classic Machine Learning algorithms using only **Python** and **NumPy**.

The goal is to understand the mathematical foundations and internal mechanics behind each algorithm, without relying on high-level machine learning libraries such as Scikit-Learn, TensorFlow, or PyTorch for the core implementations.

## Algorithms

### Implemented

- [x] Logistic Regression

- [x] Feedforward Neural Network

- [x] K-Nearest Neighbors

- [x] Naive Bayes

### Planned

- [ ] Decision Trees

- [ ] Random Forest

- [ ] Gradient Boosting Trees

- [ ] K-Means

- [ ] Support Vector Machine

## Objectives

The main objectives of this project are:

- Implement machine learning algorithms from scratch using NumPy.

- Understand the optimization process behind each model.

- Practice clean and modular Python code.

- Compare custom implementations with standard machine learning libraries.

- Build a strong foundation in supervised and unsupervised learning.

## Tech Stack

- Python

- NumPy

- Matplotlib, for visualization

- Jupyter Notebook, for experiments and explanations
